# My Dotfiles for a Legendary Setup

Herein lies all the tools I use for my daily workflows. They are optimised for
me, but I am always in search of better ways to improve the efficiency and
productivity of my workflows.

# Notable Mentions

Neovim - Highly customisable editor that uses Vim Motions to navigate text and
files.

Wezterm - Highly customisable terminal emulator pack-filled with useful features
(e.g. workspaces).

Oh-My-Posh - For a beautiful terminal prompt.

Aerospace - Tiling manager for macOS, make switching and tiling apps just a
keymap away!

Television - A powerhouse of a CLI fuzzy-finder (built in Rust, and just as fast
as `fzf`).

Sioyek - Keyboard-focused PDF viewer.

Yazi - Terminal-based file manager.
